# My QA Assignment

Playwright tests for SauceDemo (UI) and ReqRes (API).

## Getting started

```bash
cd my-qa-assignment
npm install
```

Create a file named `.env` in the root folder and paste the content from `.env.template`.

Update the values in `.env` if you need to.

`.env` is gitignored. Specs also have fallbacks, so you can still run without one.

## Running tests

```bash
npx playwright test
```
